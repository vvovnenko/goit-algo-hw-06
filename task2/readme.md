### Граф
```
{
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F', 'E'],
    'D': ['B', 'E'],
    'E': ['B', 'C', 'F', 'D'],
    'F': ['C', 'E']
}
```

### DFS шлях:
A B D E C F

### BFS шлях
A B C E D F

### Висновки
Шлях обходу виршин відрізняється тому що BFS відвідує 
всі вершини на певному рівні перед тим, як перейти до наступного рівня. 
В свою чергу DFS відвідує весь шлях перед тим як перейти до іншого
